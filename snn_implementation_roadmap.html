Spiking Neural Network implementation roadmap across 6 phases

1
Neuron model fundamentals weeks 1–2
Implement the Leaky Integrate-and-Fire (LIF) neuron from scratch in NumPy. Understand membrane potential, threshold firing, and reset dynamics before touching any library. This is your foundation — don't skip it.
LIF neuron ↗ Membrane potential ↗ Spike threshold ↗ NumPy impl ↗
2
Spike encoding schemes weeks 2–3
Learn to convert real-valued data (images, signals) into spike trains. Implement rate coding (spike frequency ∝ input magnitude) and temporal coding (spike timing encodes value). This is the SNN equivalent of preprocessing.
Rate coding ↗ Temporal coding ↗ MNIST encoding ↗ Poisson spikes ↗
3
SpikingJelly or Norse setup weeks 3–4
Migrate from raw NumPy to a proper SNN framework. SpikingJelly (PyTorch-based, Chinese Academy of Sciences) is the most mature choice. Build a simple SNN layer, connect neurons, and reproduce your Phase 1 LIF behavior in the framework.
SpikingJelly setup ↗ Norse alt ↗ First SNN layer ↗ Surrogate gradient ↗
4
STDP learning rule weeks 4–5
Implement Spike-Timing-Dependent Plasticity — the biologically plausible learning rule that strengthens synapses when pre-synaptic spikes precede post-synaptic ones. This is the crown jewel of SNN research. Also implement backprop via surrogate gradients as a modern alternative.
STDP rule ↗ STDP impl ↗ Backprop via surrogates ↗ Hebbian link ↗
5
Benchmark on MNIST / N-MNIST weeks 5–6
Train your SNN on MNIST (static) and N-MNIST (neuromorphic, event-based — like MNIST but captured with a dynamic vision sensor). Compare accuracy vs synaptic operations count vs energy estimate. Hitting ~99% on MNIST and ~98% on N-MNIST is the target. These numbers go in your README.
N-MNIST dataset ↗ Energy metric ↗ SynOps count ↗ MNIST baseline ↗
6
Neuromorphic hardware sim + writeup weeks 7–8
Simulate how your SNN would behave on Intel Loihi-style neuromorphic hardware — event-driven, sparse, near-zero power when silent. Profile spike sparsity and estimate power consumption vs an equivalent ANN. Finish with a README and a blog post comparing SNN vs ANN on every axis: accuracy, speed, energy, and interpretability.
Loihi sim ↗ Spike sparsity ↗ Power estimate ↗ Writeup guide ↗
